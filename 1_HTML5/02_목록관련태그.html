목록 관련 태그

u1 태그 : 순서 없는 목록 (Unordered List)
ol 태그 : 순서 있는 목록 (Ordered List)
li(List item) 태그 : ul, ol 태그의 내부에 작성되는 태그로 목록의 요소를 나타냄.

ol 태그

* ol 태그의 type 속성 *
type="1" : 숫자 (기본값)
type="a" : 소문자 알파벳
type="A" : 대문자 알파벳
type="i" : 소문자 로마 숫자
type="I" : 대문자 로마 숫자
start 속성 : 목록의 시작 번호를 지정하는 속성
reversed 속성 : 역순으로 번호 매기기(속성만 사용 가능)
