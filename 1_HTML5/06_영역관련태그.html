영역을 나누는 형식(성질)

        
block 형식

        - 항상 새로운 줄에서 시작
        - 이전 요소의 아래에 위치(블럭)
        - 크기를 지정하는 width, height 속성을 사용할 수 있음
    
        ex) p/pre(문단 나누기), h1 ~ h6
        hr (수평선 사용하여 화면 나누기), ul, ol, li(목록)
        div (영역 나누기)
    
        
inline 형식

        - 동일한 줄에 다른 inline 요소와 나란히 배치
        - 크기를 지정하는 width, height 속성을 사용할 수 없음(콘텐츠의 너비만큼 차지)
        ex) a(링크), img(이미지), span(영역 나누기)
        strong(굵게), em(기울임), br(줄바꿈)

block(div) / inline(span) 차이점 1 : 줄바꿈(개행)
